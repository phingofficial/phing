B.37 ReflexiveTask

The ReflexiveTask performs operations on files. It is essentially a convenient way to transform (using filter chains) files without copying them.

Table B.37: Attributes

NameTypeDescriptionDefaultRequired
fileStringA single file to be processed.n/aYes (unless <fileset> provided)

B.37.1 Examples

<reflexive>
  <fileset dir=".">
    <include pattern="*.html">
  </fileset>
  <filterchain>
    <replaceregexp>
      <regexp pattern="\r(\n)" replace="\1"/>
    </replaceregexp>
  </filterchain>
</reflexive>

B.37.2 Supported Nested Tags:

  • fileset

  • filterchain
