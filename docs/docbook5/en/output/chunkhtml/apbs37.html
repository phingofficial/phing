B.37 WaitForTask

Wait for a condition to become true or a timeout, whichever comes first.

Table B.39: Attributes

NameTypeDescriptionDefaultRequired
MaxWaitIntegerSet the maximum length of time to wait in units3minYes
WaitUnitStringSet the max wait time unit. Must be one of "week","day", "hour", "minute", "second", "millisecond"millisecondNo
CheckEveryIntegerSet the time between each check500msYes
CheckEveryUnitStringSet the check every time unit. Must be one of "week","day", "hour", "minute", "second", "millisecond"millisecondsNo
TimeoutPropertyStringName of the property to set after a timeout. nullNo

B.37.1 Examples

Wait for a maximum of ten seconds for the file "ready" to appear.

<waitfor maxwaitunit="second" maxwait="10"> 
  <available file="ready"/>
</waitfor>

B.37.2 Supported Nested Tags

All conditionals including and, or, not etc.
