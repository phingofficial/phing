B.26 LoadFileTask

The LoadFileTask loads the contents of a (text) file into a single property.

Table B.27: Attributes

NameTypeDescriptionDefaultRequired
propertyStringThe name of the property to set.n/aYes
file (or srcFile)StringThe file to load.n/aYes
failonerrorBooleanWhether to halt the build on failure.trueNo
quietBooleanDo not display a diagnostic message (unless Phing has been invoked with the -verbose or -debug switches) or modify the exit status to reflect an error. Setting this to true implies setting failonerror to false.falseNo

B.26.1 Examples

<loadfile property="version" file="version.txt"/>

B.26.2 Supported Nested Tags:

  • filterchain
